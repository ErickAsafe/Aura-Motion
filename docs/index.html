Motion Graphics com I.A.

O Aura Motion é um framework que une a renderização matemática do Remotion com a fluidez infinita do GSAP. Totalmente orquestrado por agentes autônomos.

A Arquitetura Híbrida

O maior desafio da engenharia de motion em web é que bibliotecas maduras como o GSAP rodam em tempo real (baseado no relógio interno do navegador), mas gravadores profissionais como o Remotion renderizam offline (processam um frame a cada alguns segundos).

A solução do Aura Motion é o "Truque do Tempo". Nós escrevemos o GSAP, mas não deixamos ele rodar. Nós pausamos a timeline e forçamos o seeker a avançar matematicamente lendo a câmera (frame) do Remotion. O código gerado pela IA sempre respeita esse padrão: 

import { useCurrentFrame, useVideoConfig } from 'remotion';
import { useEffect, useMemo, useRef } from 'react';
import gsap from 'gsap';

export const CenaHibrida = () => {
  const frame = useCurrentFrame();
  const { fps } = useVideoConfig();
  const boxRef = useRef(null);

  // 1. O agente constrói a timeline pausada
  const tl = useMemo(() => {
    const t = gsap.timeline({ paused: true });
    t.to(boxRef.current, { x: 500, duration: 2, ease: "power4.out" });
    return t;
  }, []);

  // 2. A sincronização mestre do Aura Motion
  useEffect(() => {
    tl.seek(frame / fps); // Força o GSAP a sincronizar com a gravação offline
