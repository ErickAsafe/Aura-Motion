## Pipeline Orquestrada por IA
O framework usa 4 agentes para roteirizar, buscar referências e programar animações web automaticamente.

## Renderização via Puppeteer/FFmpeg
As animações são feitas em HTML puro e GSAP, congelando o tempo frame a frame para gravar em MP4 sem lag.

## Metáforas Visuais Contínuas
Nenhuma cena fica estática. O framework exige loops contínuos de animação (respiração) em cada frame.
